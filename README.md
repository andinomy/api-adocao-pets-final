# API Adoção de Pets 🩷🐈🐕

Projeto desenvolvido para a disciplina de Desenvolvimento de Serviços Web.

## Descrição

API RESTful para gerenciamento de adoção de animais de estimação.

O sistema permite:

* Cadastro de usuários
* Cadastro de pets
* Login com autenticação JWT
* Adoção de pets
* Controle de permissões por perfil (admin e adopter)

## Tecnologias

* Node.js
* Express
* MySQL
* JWT
* bcrypt
* ESLint
* Prettier
